# CapyDex

An index that stores information of items in the game Capybara Go

## What This Site Contains
Information on the following systems:
1. Adventurers
2. Artifacts
3. Brands
4. Equipments
5. Gems
6. Heroes
7. Mounts
8. Mythic Treasure (known as Relics)
9. Pets
10. Pet Armaments
